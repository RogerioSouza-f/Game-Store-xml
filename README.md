# Loja de Jogos 🎮

Um aplicativo Android moderno para gerenciamento de uma loja e biblioteca de jogos, desenvolvido com as melhores práticas de desenvolvimento mobile.

## ✨ Funcionalidades

### 🛒 Loja
*   **Destaques**: Banner interativo com o jogo do momento.
*   **Catálogo**: Lista completa de jogos com preços, categorias e avaliações.
*   **Detalhes**: Informações completas sobre cada título, incluindo descrição e desenvolvedora.

### 📚 Biblioteca Personalizada
*   **Aba Todos**: Veja todos os jogos que você adquiriu.
*   **Aba Instalados**: Gerencie o que está pronto para jogar.
*   **Aba Favoritos**: Acesso rápido aos seus jogos marcados com a estrela.

### ⚙️ Ciclo de Vida do Jogo
*   **Compra**: Adquira jogos diretamente na loja.
*   **Instalação**: Gerencie o estado de instalação (Instalar/Desinstalar) para simular o uso de disco.
*   **Favoritos**: Marque seus jogos preferidos com feedback visual dinâmico (Estrela Azul).

### 👤 Perfil e Configurações
*   **Editar Perfil**: Altere seu nome e e-mail com persistência em memória durante a sessão.
*   **Configurações**: Opções de notificações e modo escuro (UI).

## 🚀 Tecnologias Utilizadas

*   **Linguagem**: [Kotlin](https://kotlinlang.org/) (JVM 11)
*   **Arquitetura**: XML + Fragments com Single Activity.
*   **Navigation Component**: Gestão de rotas e passagem de argumentos (Safe Args).
*   **View Binding**: Acesso seguro aos componentes de UI.
*   **Material Design 3**: Interface moderna e responsiva.
*   **Coil**: Carregamento de imagens assíncrono e otimizado.
*   **RecyclerView**: Listagem eficiente de dados.

## 🛠️ Como rodar o projeto

1.  Clone este repositório.
2.  Abra no **Android Studio Hedgehog** ou superior.
3.  Certifique-se de que o JDK 11 está configurado.
4.  Sincronize o Gradle.
5.  Execute no seu emulador ou dispositivo físico!
